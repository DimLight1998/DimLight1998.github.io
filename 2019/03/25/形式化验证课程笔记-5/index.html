Theory of Equality and Uninterpreted Functions

In $T_E$，$=$ is a predicate. Formulas in $T_E$ may contain other predicates, but they can be converted to $=$ using Theory of Equality and Uninterpreted Functions (EUF).

Any predicate $p(x_1, x_2, \dots, x_n)$ other than $=$ can be converted to a function and the predicate $=$, where $f_p$ is a introduced function by $p$ and $\bullet$ is a constant.

In EUF:

  • $=$ is the only predicate.
  • All atomic formulas are either equalities or inequalities.

Relations

  • A relation on $S$ is a subset of $S\times S$.

  • An equivalence relation should be reflexive, symmetric and transitive.

  • An congruence relation $R$ should be a equivalence relation, and obey the property that $\forall \mathbf s, \mathbf t\in S^n::(\land^n_{i = 1} \mathbf s_iR\mathbf t_i)\rightarrow f(\mathbf s)Rf(\mathbf t)$, where $f:S^n\mapsto S$ is not arbitrary. (A relation is a congruence relation relies on the function $f$.)

Classes

  • A equivalence class for equivalence relation $R$ for element $e$ is $[e]_R = \{s| s\in S \land s R e\}$.
  • If the relation is also a congruence relation, the equivalence class is also a congruence relation.

Partitions

  • A partition $P$ of a set $S$ is a set of subsets of $S$ with total property and disjoint property.

  • All equivalence classes for relation $R$ on $S$ form a partition, called the quotient of $S$ by $R$: $S/R$.

Relation Refinements

  • Refinement: $R_1 \prec R_2$ if $\forall s_1, s_2 \in S:: s_1 R_1 s_2 \rightarrow s_1 R_2 s_2$, called $R_1$ refines $R_2$.
  • Refinements and subsets: $R_1\prec R_2$ if and only if $R_1 \subseteq R_2$.

Example:

Define $a\equiv_n b$ if $(a\text{ mod } n = (b \text{ mod } n))$, then we have $\equiv_4 \prec \equiv_2$.

Equivalence Closure and Congruence Closure

The equivalence closure for any relation $R$ is the smallest equivalence relation $R_E$ including $R$.

Formally, $R_E$ is the equivalence closure of $R$ if:

  • $R\prec R_E$.
  • For any equivalence relation $R'$, $R\prec R'\rightarrow R'=R_E \lor R_E\prec R'$.

Congruence closures can be defined similarly.

Deciding Quantifier-free Conjunctive Segment of $T_E$

Basic Idea

For any quantifier-free conjunctive $T_E$ formula $F:(s_1=t_1)\land (s_2=t_2)\land\dots\land(s_m=t_m)\land(s_{m + 1}\ne t_{m + 1})\land\dots\land(s_n\ne t_n)$, $F$ is $T_E$-satisfiable if and only if there is a congruence relation $\sim$ such that:

  • $s_i\sim t_i$ for all $i\in\{1,\dots, m\}$
  • $s_i\nsim t_i$ for all $i \in \{m + 1, \dots, n\}$

Algorithm

  1. Construct a relation $R$ based on $F$ over the subterm set $S_F$(variables) : $\{s_1 = t_1, \dots ,s_m=t_m\}$.
  2. Construct the congruence closure $\sim$ of $R$.
  3. If for any $i\in \{m + 1, \dots, n\}$, $s_i\sim s_t$, then $F$ is $T_E$-unsatisfiable.
  4. Otherwise it's $T_E$-satisfiable.

Congruence Closure Algorithm

Algorithm

The congruence closure over $S_F$ can be constructed as:

  1. Construct a trivial congruence closure $\sim_0 = \{\{s\}|s\in S_F\}$.
  2. For any $i \in \{1, \dots, m\}$, we have $s_i = t_i$, so build $\sim_{i}$ by merging $[s_i]_{\sim_{i - 1}}$ and $[t_i]_{\sim_{i - 1}}$ and then propagate until you can't perform any more merging or propagating.

Example

For formula $F: f(a,b) = a\land f(f(a, b), b)\ne a$:

  1. Get $S_F = \{a, b, f(a, b), f(f(a, b), b)\}​$.
  2. $\sim_0 = \{\{a\}, \{b\}, \{f(a, b)\}, \{f(f(a, b), b)\}\}$.
  3. Based on $f(a, b) = a$ we have $\sim_{0}' = \{\{a, f(a, b)\}, \{b\}, \{f(f(a, b), b)\}\}$.
  4. With axiom of $T_E$, we have $f(a, b)\sim a\land b\sim b \rightarrow f(f(a, b), b)\sim f(a, b)$, hence $\sim_0'$ can be propagated as $\sim_1 = \{\{a, f(a, b), f(f(a, b), b)\}, \{b\}\}$.
  5. We have $a\sim f(f(a, b), b)$ but $F$ says $f(f(a, b), b) \ne a$, so $F$ is not $T$-satisfiable.

Property of the Algorithm for Deciding Satisfactory of $T_E$

Soundness and Completeness

Quantifier-free conjunctive $\Sigma_E$ formula $F$ is $T$-satisfiable if and only if the algorithm returns satisfiable.

Complexity

The time complexity is $O(e^2)$ where $e$ is the number of edges in the initial graph of the graph representing the congruence relation.

Deciding Quantifier-free Conjunctive Segment of $T_A$

Basic Idea

The problem can be reduced to deciding quantifier-free conjunctive segment of $T_E$.

Case 1: Formulas with No Write Terms

In this case, all terms are reading of array. For each array $a$, we can introduce a function $f_a$. For the reading term $a[i]$, it can be rewritten into a function $f_a(i)$. Now the formula is converted to a formula in $T_A$.

Case 2: Formulas with Write Terms

Case 2 can be reduced to case 1.

Select one write term from the formula $F$, then convert it as following:

  1. (Case $i = j$) Replace $F[a\langle i\lhd v\rangle[j]]$ with $F_1: F[v] \land i = j$.
  2. (Case $i\ne j$) Replace $F[a\langle i\lhd v\rangle [j]]$ with $F_2: F[a[j]]\land i\ne j$.

Now we have two branches and reduce one write term. Do this recursively on each branch until there is no write terms in the formula, then every formula has been transformed into case 1. If any branch is satisfiable, then the original formula is satisfiable. Otherwise, the original formula is unsatisfiable.

Properties

This algorithm is sound and complete. The time complexity is $O(2^n)$.
