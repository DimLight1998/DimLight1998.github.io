Happy 与 Parsec 学习笔记

Parser Generator 与 Parser Combinator

Parser 有两种——Generator 和 combinator。

  • Generator 接受语法规则，然后生成语法分析程序。例如 ANTLR 就是典型的 parser generator。

  • Combinator 将一些小的语法分析函数组合起来，形成功能更复杂的语法分析函数。

Happy 和 Parsec 是 Haskell 的两个包，前者是 parser generator，后者是 parser combinator。

作为例子，这篇文章会使用两种不同的策略对 json 文件进行语法分析。
