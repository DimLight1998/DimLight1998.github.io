第四周

Review FOL

Logical Symbols and Non-logical Symbols

  • Logical symbols: propositional connectives, variables, quantifiers.
  • Non-logical symbols: constant symbols, function symbols, predicate symbols.

Interpretation and Assignment

  • Interpretation fixes values for constants.
  • Assignment fixes values for variables.

First-Order Theory: Motivation

First-order logic is very general but undecidable. By limiting the non-logical symbols, we can have something stronger then propositional logic but still decidable, which is first-order theory. First-order theory is weaker then FOL.

Defining a First-Order Theory

A first-order logic contains:

  • Signature $\Sigma$, a set of non-logic symbols
  • Axioms $\mathcal{A}$, a set of closed formulas over $\Sigma$

$\Sigma$ limits the symbols and $\mathcal A$ gives them semantics.

Formula only containing symbols from $\Sigma$ is called $\Sigma​$-formula.

Terminology

  • T-interpretation/T-model: A tuple of domain and interpretation $(D, I)$ satisfying $\forall A\in \mathcal A, (D, I)\vDash A$. (Domain and interpretation satisfying all axioms.)
  • T-satisfaction: $F$ is T-satisfiable if you can find a T-model $T$ and an assignment $\alpha$ such that $T, \alpha\vDash F$.
  • T-validity: Similar as above, but asks for all T-models and assignments. Written as $T\vDash F$. A theory is the set of all closed formulas that are T-valid.
  • T-entailment: $F_1$ T-entails $F_2$ if $T\vDash F_1\rightarrow F_2$.
  • T-equivalence: Similar as above but asks for $F_1\leftrightarrow F_2$.

Proving

If you want to prove validity of some formula of a first-order theory, you can use FOL rules and axioms from the theory.

Theory of Equality $T_E$

  • Signature: $\Sigma_E = \{=, a, b, \dots, p, q, r, \dots\}​$
  • Axioms:
    • Reflexivity: $\forall x :: x = x​$
    • Symmetry: $\forall x, y::x = y\rightarrow y = x​$
    • Transitivity: $\forall x, y, z:: x= y\land y - z\rightarrow x = z​$
    • Function congruence:$\forall \vec x, \vec y::\vec x_i =\vec y_i \rightarrow f(\vec x) = f(\vec y)​$
    • Predicate congruence:$\forall \vec x, \vec y::\vec x_i =\vec y_i \rightarrow (p(\vec x) \leftrightarrow p(\vec y))$

  • Not decidable. FOL is not weaker than $T_E$.

  • Quantifier free fragment of $T_E$ is decidable.
