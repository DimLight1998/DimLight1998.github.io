第六周

Combination Theories

Combined Theory

For theory $T_1, T_2$, if $T_1\cap T_2 = \{=\}$, then the combined theory $T_1\cup T_2$ has signature $\Sigma_1\cup\Sigma_2$ and axioms $A_1\cup A_2$.

Nelson-Oppen Combination Method

$T_1\cup T_2$ is decidable if both $T_1$ and $T_2$ are quantifier-free, conjunctive fragments which are decidable and stably-infinite (see below).

Stably-Infinite Theories

A theory $T$ with signature $\Sigma$ is stably infinite if for every quantifier-free $T$-satisfiable $\Sigma$-formula $F$, there is a satisfiable $T$-interpretation with domain of infinite cardinality.

$T_E$, $T_A$ and $T_Z$ are stably infinite.

Nelson-Oppen Procedure

This procedure takes a formula $F$ in combined theory $T_1\cup T_2$, and returns an equisatisfiable formula $F_1\land F_2$ where $F_1$ is a $T_1$-formula and $F_2$ is a $T_2$-formula.

In short, the procedure can separate a formula in a combined theory into two formulas in each theory.

Step 1. Variable Abstraction

Procedure

In this step, our goal is to make every literal in $F$ belongs to either $T_1$ or $T_2$.

A function or a predict is a literal, so:

  • If function $f(\dots, t, \dots)$ is in $F$, where $f$ in $\Sigma_1$ but $t$ in $\Sigma_2$, then we introduce a new variable $w$ and replace $t$ with w, then conjoin $t = w$.
  • If predicate $p(\dots, t, \dots)$ is in $F$, where $p$ in $\Sigma_1$ but $t$ in $\Sigma_2$, we introduce a new variable $w$ and replace $t$ with w, then conjoin $t = w$.

Note that a variable is a logic symbol. When finished, every literal in $F$ belows to either $T_1$ or $T_2$. Note the $=$ cannot be viewed as a predicate here.

Step 2. Guess And Check

Procedure

In this step, our goal is to decide whether the original formula is satisfiable.

  1. For given formula $F_1$ and $F_2$, we can find their shared variables $V = free(F_1)\cap free(F_2)$, where $free(F)$ means all free variables in $F​$.

  2. Build an equivalence relation $E$ on $V$, and build a formula $\alpha(V, E)$ which depends on $V$ and $E$:

  3. $F_1\cup F_2$ is $T_1\cup T_2$-satisfiable if and only if there exists an $\alpha(V, E)$ and:

    • $F_1\land \alpha(V, E)$ is $T_1$-satisfiable.
    • $F_2\land \alpha(V, E)$ is $T_2$-satisfiable.

Example

(Follows the example in step 1.)

  1. $F_1$ (which in $T_E$) is $f(x)\ne f(m)\land f(x)\ne f(n)$, and $F_2$ (which in $T_\mathbb Z$) is $1\le x \land x \le 2 \land m = 1\land n = 2$.
  2. Shared variables are $V = \{x, m, n\}$.
  3. For every possible partition on $V$, we can construct an equivalence relation. There are five possibilities.
    • $\{\{x\}, \{m\}, \{n\}\}$: No, because $F_2\land x\ne m \land x \ne n$ is not $T_\mathbb Z$-satisfiable. (Some items are omitted.)
    • $\{\{x\}, \{m =n\}\}$: No, because $F_2\land m=n$ is not $T_\mathbb Z$-satisfiable.
    • $\{\{x, m\}, \{n\}\}$: No, because $F_1\land x = m$ is not $T_E$-satisfiable.
    • Same for the remaining two possibilities.
  4. As all possibilities cannot ensure $F_1\land \alpha(V, E)$ is $T_E$-satisfiable and $F_2 \land \alpha(V, E)$ is $T_\mathbb Z$-satisfiable, the original formula $F$ is not $T_E\cup T_\mathbb Z$-satisfiable.

Efficiency Issue

The method above is impractical as we need to construct too many partitions.

Convex Theory

A theory is convex if for any quantifier-free, conjunctive formula $F$:

  • $T_A$ and $T_\mathbb Z$ are not convex.
  • $T_E$ and $T_\mathbb Q$ are convex.

$T_\mathbb Z$ is not convex, for example, for formula $F:1 \le x\le 2 \land u=1 \land v = 2 $, we have $F\rightarrow x=u\lor x=v$ is true, but both $F\rightarrow x=u$ and $F\rightarrow x = v$ are false.

Optimized Step 2: Equality Propagation

Procedure

In this step, our goal is to decide whether the original formula is satisfiable, but with a more efficient algorithm than the step 2 above.

  1. For given formula $F_1$ and $F_2$, we can find their shared variables $V = free(F_1)\cap free(F_2)$, where $free(F)$ means all free variables in $F$.
  2. Construct a set $E$ containing all equations shared between $T_1$ and $T_2$. Initially the set is empty.
  3. Ask the decision procedure of $T_1$ to report newly implied equalities between shared variables. If $T_1$ is convex, you will get one equality, and you should add this equality to $E$; if $T_1$ is not convex, you will get a disjunction of equalities, and you should split the search procedure into multiple branches and add each equality into one branch.
  4. Repeat step 3, but replace $T_1$ with $T_2$ and use information stores in $E$.
  5. Repeat step 3 and step 4, until
    • If any branch of the search ends with a full arrangement, the original formula is satisfiable.
    • If all branches end with contradiction, the original formula is unsatisfiable.
    • If all branches cannot discover new equalities, the original formula is satisfiable.

Example

Formula $F$ is in $T_E\cup T_\mathbb Q$:

Divide it into two formulas:

  • $F_E: f(w) \ne f(z) \land u=f(x)\land v =f(y)$
  • $F_\mathbb Q: x\le y \land y + z \le x \land 0 \le z \land w = u - v$
  • $V = \{x, y, z, u, v, w\}$

If we use the procedure in the above section by constructing $E$ on all partitions of $V$, it is impractical.

Note that both $T_E$ and $T_\mathbb Q$ are convex.

  1. Construct known equivalence relation $E​$ on $V​$, but initially the set is empty.
  2. With $F_\mathbb Q$ and $E=\{\}$, we now know $x = y$. Add this equation to $E$. (Although we know $z=0$, $0$ is not in $T_E$, so we cannot add this equation to $E$. We only concern about the equality on shared variables.)
  3. With $F_E$ and $E=\{x = y\}$, we now know $u=v$. Add this equation to $E$.
  4. With $F_\mathbb Q$ and $E = \{x = y, u = v\}$, we now know $z = w$. Add this equation to $E$.
  5. With $F_E$ and $E$ we get a contradiction, hence the original formula is $T_E\cup T_\mathbb Q$-unsatisfiable.

How to construct an assignment: if the formula is satisfiable, you can construct an assignment by defining equivalence relations on $V$: if two variables are equal according to $E$, they are equal; otherwise they are not equal.

Another Example

Formula $F$ is in $T_E\cup T_\mathbb Z$:

Divide $F$ into two formulas:

  • $F_E: f(x)\ne f(w_1)\land f(x)\ne f(w_2)$
  • $F_\mathbb Z: 1\le x \land x \le 2 \land w_1 = 1 \land w_2 = 2$
  • $V = \{x, w_1, w_2\}$

Note that $T_\mathbb Z​$ is not convex.

  1. Construct known equivalence relation $E$ on $V$, but initially the set is empty.
  2. With $F_\mathbb Z$ and $E = \{\}$, we now know $x = w_1\lor x = w_2$, and we have two path here: $E_1 = \{x = w_1\}$ and $E_2 = \{x = w_2\}$.
  3. With $F_E​$ and $E_1 = \{x = w_1\}​$ we get a contradiction, hence this path is not satisfiable.
  4. With $F_E​$ and $E_2 = \{x = w_2\}​$ we get a contradiction, hence this path is not satisfiable.
  5. Therefore the original formula is not $T_E\cup T_\mathbb Z$-satisfiable.

Bring Back Quantifiers

Quantifier-free fragments are not expressive, for example, we cannot say two arrays are equal in $T_{A}$.

Array Properties

Introduce quantifiers into $T_A$, but limit the variable to array indices.

An array property is a $\Sigma_A$-formula of the form $\forall i::F[i]\rightarrow G[i]$, where $F[i]$ is index guard, $G[i]$ is value constraint.

Syntax of Value Constraint

Array Property Fragment

The array property fragment is a logical combination of quantifier-free $\Sigma_A$-formulas and array properties.

Examples

  • $\forall i::a[i] = b[i]$
  • $\forall j::i\ne j\rightarrow a\langle i\triangleleft v\rangle[j] = a[j]$

Deciding Array Property Fragment

Semantically, $\forall i::F[i]$ is equal to $\bigwedge\limits_\beta \beta (F)$ where $\beta$ ranges over all substitutions of $i$.

The idea is to replace $\forall i::F[i]$ with a finite conjunction $\bigwedge\limits_{i = 1}^n F[t_i]$. In this conjunctive, $t_i$ are called the index terms, and all index terms necessary should be considered.
