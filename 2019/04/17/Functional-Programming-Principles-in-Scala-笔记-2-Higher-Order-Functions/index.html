构造函数

对于每个类，Scala 会自动生成一个 primary constructor。它的参数是类的参数，执行该构造函数时会顺次执行类的定义中的语句，这就是为什么 require 会被执行。

也可以自定义一些构造函数。关键字是 this。构造函数本身还可以调用其它构造函数。

操作符

对于只有一个参数的函数，. 和括号都可以省略，看起来就会像中缀运算符。也就是说你可以把 a.add(b) 直接写成 a add b。

Scala 中的标识符的规则和其它语言不一样，它是可以含有符号的。一个标识符可以：

  • 像其他语言一样，字母（或下划线）开头，然后是字母（或下划线）和数字的序列；
  • 完全由符号构成的序列，例如 +?%&；
  • 由第一条规则构造出的序列，接一个下划线，再接一个由第二条规则构成的序列。

优先级

对于自己定义的这些操作符，优先级是如何确定的？每个操作符的第一个字符反映了该操作符的优先级。Scala 对不同字符定义了不同优先级。例如 * 的优先级比 + 高，因此 a *$% b +@#$ c *@#$ d 的实际顺序是 (a *$% b) +@#$ (c *@#$ d)。

a + b ^? c ?^ d less a ==> b | c 的求值顺序是 ((a + b) ^? (c ?^ d)) less ((a ==> b) | c)。特殊符号的优先级最高，字母的优先级最低。中间的那些字符的优先级和其他语言中常使用的惯例相同。
