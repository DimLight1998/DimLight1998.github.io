其它线性结构

Vector

实现上，Vector 是一棵很浅的树，每个结点可能有 32 个子结点，这样对缓存是比较友好的。

Vector 的随机访问 可以看作是 常数时间的，实际上增长非常缓慢。

Vector 适用于需要批量访问的场景，例如 map 操作。

Range

Range 表示整数的一个范围。使用 to 表示包括上界，until 不包括上界，可使用 by 指定步长。

Scala 中有个叫 Seq 的类型。List、Vector 和 Range 都是它的子类型。还有个叫 IndexedSequence 的类，Vector 和 Range 都是它的子类。IndexedSequence 也是 Seq 的子类。

线性结构可以进行的操作

除了前面提到的列表上的操作外，还有：

  • exists 和 forall 量词
  • zip 和 unzip
  • flatMap：将线性结构中的每个元素映射为一个 collection，然后将这些 collection 合并；也可以先进行 map 然后执行 flatten
  • sum 和 product 求和、求积
  • max 和 min 取出最大元素和最小元素
  • sorted 按自然序进行排序，sortBy 指定排序依据
  • groupBy 可以按指定方法进行分组，会得到一个 Map

实际上上述方法对 Iterable 都是可以使用的。

Map

Scala 中的 Map 即为映射。

经过我的测试，a -> b 实际上就是 (a, b)。

有趣的是，Map[A, B] 是 A => B 的子类型。很多使用函数的地方都可以直接传入一个 Map 进去。如果希望在键不存在的时候返回特殊值而不是抛异常，可以使用 get 函数，它会返回 Option[B]。

对一个 Map 使用 withDefaultValue 可以将其转化为一个 total 的函数。它对不存在的键会返回默认值。
