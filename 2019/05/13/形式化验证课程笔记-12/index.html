形式化验证课程笔记 12

发表于 2019-05-13

第十二周

抽象解释

抽象解释 是对语义进行抽象分析的过程。它会将程序的解释从具体的空间映射到一个抽象空间中，可以用来搞静态分析。

数学基础

偏序

偏序关系是自反、传递、反对称的关系。

偏序集记作 $\displaystyle{(D, \sqsubseteq)}$。

格

如果一个偏序集满足：

  • 任意两个元素 $\displaystyle{a, b}$ 都可以导出一个最小上界 join：$\displaystyle{a\sqcup b}$
  • 任意两个元素 $\displaystyle{a, b}$ 都可以导出一个最大下界 meet：$\displaystyle{a\sqcap b}$

这样的偏序集，以及对应的 join 和 meet 定义构成了一个 格。

是偏序但是不是格的例子，其中 a 和 b 找不到 join 和 meet

完全格

对于一个格，如果它的所有元素构成的集合的 每个子集都有自己的最小上界和最大下界，则称为 完全格。完全格具有最小元 $\displaystyle{\bot = \sqcup\varnothing}$ 和最大元 $\displaystyle{\top = \sqcup D}$，其中 $\displaystyle{\sqcup X}$ 表示 $\displaystyle{X}$ 的最小上界，$\displaystyle{D}$ 是格的所有元素构成的集合。

链

链 是偏序集的子集，其中任意两个元素应当可以进行比较。

完全偏序

如果偏序集满足：

  • 具有最小元 $\displaystyle{\bot_D}$
  • 任意一个链 $\displaystyle{S}$ 都具有最小上界 $\displaystyle{\sqcup S}$（这个最小上界不一定要在 $\displaystyle{S}$ 中，但我不确定）

则这个偏序集为 完全偏序。

偏序但不是完全偏序的例子是具有自然序的自然数。因为存在无穷长的链，这样的链中不存在最小上界。不过可以通过添加 $\displaystyle{\infty}$ 元素来让集合变成是完全偏序的。

函数

单调性

对于偏序集 $\displaystyle{(D, \sqsubseteq)}$ 和 $\displaystyle{(D', \sqsubseteq')}$，若函数 $\displaystyle{f: D\mapsto D'}$ 满足 $\displaystyle{x\sqsubseteq y \rightarrow f(x)\sqsubseteq'f(y)}$，称其为 单调的。

连续性

对于完全偏序的偏序集 $\displaystyle{(D, \sqsubseteq)}$ 和 $\displaystyle{(D', \sqsubseteq')}$，若函数 $\displaystyle{f: D\mapsto D'}$ 满足：对于 $\displaystyle{D}$ 中的每条链 $\displaystyle{S}$，$\displaystyle{f}$ 操作和 $\displaystyle{\sqcup}$ 操作是可交换的（即 $\displaystyle{\sqcup f(S)}$ 存在且 $\displaystyle{\sqcup f(S) = f(\sqcup S)}$），则称其为 $\displaystyle{\sqcup}$-连续的。

单调但是不连续的例子：$\displaystyle{f:\mathbf N\cup \{\infty\}\mapsto \{0, 1\}}$，$\displaystyle{f}$ 将 $\displaystyle{\infty}$ 映射为 $\displaystyle{1}$，其它映射为 $\displaystyle{0}$。

单调同时连续的例子：修改上一个例子，将小于某个值的所有数映射为 $\displaystyle{0}$，其它映射为 $\displaystyle{1}$。

不动点

对于定义在偏序集 $\displaystyle{(D, \sqsubseteq)}$ 上的函数 $\displaystyle{f}$，以及元素 $\displaystyle{x}$：

  • 若 $\displaystyle{f(x) = x}$，则 $\displaystyle{x}$ 为 不动点
  • 若 $\displaystyle{x \sqsubseteq f(x)}$，则 $\displaystyle{x}$ 为 前不动点
  • 若 $\displaystyle{f(x) \sqsubseteq x}$，则 $\displaystyle{x}$ 为 后不动点

$\displaystyle{x}$ 可以不是上述三种情况之一，因为 $\displaystyle{f(x)}$ 和 $\displaystyle{x}$ 不一定可以比较。

最小（大）不动点

函数的所有不动点中，最小的和最大的分别称为 最小不动点（lfp） 和 最大不动点（gfp）。

不动点理论

Tarski 不动点定理

完全格上的单调函数的不动点构成了一个非空的完全格。此外，

  • 该单调函数的最小不动点是函数所有后不动点的最大下界；
  • 该单调函数的最大不动点是函数所有前不动点的最小上界。

Kleene 不动点定理

对于完全偏序集 $\displaystyle{(D, \sqsubseteq)}$ 上的函数 $\displaystyle{f}$：

  • 若 $\displaystyle{f}$ 是 $\displaystyle{\sqcup}$-连续的，则最小不动点可以通过求 $\displaystyle{f^{i}(\bot), i\in \mathbf N}$ 的最小上界求出；
  • 若 $\displaystyle{f}$ 是 $\displaystyle{\sqcap}$-连续的，则最大不动点可以通过求 $\displaystyle{f^{i}(\top), i\in \mathbf N}$ 的最大下界求出。

（似乎在求 $\displaystyle{f^{i}(\bot), i\in\mathbf N}$ 时得到的序列是不断上升的？不知道这个性质是否始终成立。）

Galois 连接

对于偏序集 $\displaystyle{(C, \le)}$ 和 $\displaystyle{(A, \sqsubseteq)}$，若函数 $\displaystyle{\alpha: C\mapsto A, \gamma: A\mapsto C}$ 满足：

则称函数对 $\displaystyle{(\alpha, \gamma)}$ 是 $\displaystyle{C}$ 和 $\displaystyle{A}$ 之间的 Galois 连接。

Galois 连接的例子

在抽象解释中，$\displaystyle{C}$ 表示的是具体域（Concreate），$\displaystyle{A}$ 表示的是抽象域（Abstract）。$\displaystyle{\alpha}$ 可以进行 抽象化，而 $\displaystyle{\gamma}$ 可以进行 具体化。

例如具体域 $\displaystyle{C}$ 是整数的集合的集合 $\displaystyle{\mathcal P(\mathbf Z)}$，抽象域是以整数为边界的区间的集合 $\displaystyle{I}$，可以定义：

  • 抽象化函数 $\displaystyle{\alpha(X) = [\min X, \max X]}$
  • 具体化函数 $\displaystyle{\gamma([a, b]) = \{n\ |\ n\in \mathbf Z\land a \le n \le b\}}$

可以验证在合适的 $\displaystyle{\le}$ 以及 $\displaystyle{\sqsubseteq}$ 定义（自然定义即可，$\displaystyle{\le}$ 是自然数的比较关系，$\displaystyle{\sqsubseteq}$ 是区间包含关系）下，$\displaystyle{(\alpha, \gamma)}$ 符合 Galois 连接的定义条件，因此它们是 Galois 连接。

在这个例子中构造出来的抽象化函数和具体化函数有些比较“感性”的性质（下面这些叙述很不严谨）：

  • 对于 $\displaystyle{\{0, 1, 4, 5\}\in C}$ 这个元素而言，将 $\displaystyle{\alpha}$ 作用在它上面时得到的是 $\displaystyle{[0, 5]}$，这是 最佳抽象（在 $\displaystyle{A}$ 中能够找到的最好的抽象）。如果修改 $\displaystyle{\alpha}$ 的定义后得到的是 $\displaystyle{[0, 10]}$，这个抽象是 可靠 的，但是没有之前那么精确。如果得到的是 $\displaystyle{[1, 4]}$，那么就是不可靠的抽象。
  • 如果最佳抽象是 $\displaystyle{a}$，那么 $\displaystyle{\gamma(a)}$ 是 $\displaystyle{C}$ 中能够被 $\displaystyle{a}$ 可靠近似的 最不精确 的元素（包含的信息量最少）。

函数抽象

在具体域 $\displaystyle{C}$ 和抽象域 $\displaystyle{A}$ 中有很多元素，两个域上可能有一些函数。抽象域上的函数可能是具体域上的函数的抽象。

对于具体域上的函数 $\displaystyle{f}$ 和抽象域上的函数 $\displaystyle{f'}$：

  • 如果 $\displaystyle{f' = \alpha\circ f\circ \gamma}$，则称 $\displaystyle{f'}$ 是 $\displaystyle{f}$ 的 最佳抽象。这是比较难得到的一种情况。
  • 如果 $\displaystyle{\forall a::(\alpha \circ f \circ\gamma)(a)\sqsubseteq f'(a)}$，则称为 可靠抽象。一个等价的条件是 $\displaystyle{\forall a::(f\circ \gamma)(a)\le (\gamma\circ f')(a)}$。
  • 如果有 $\displaystyle{\gamma\circ f' = f\circ \gamma}$，则称为 精确抽象，这是最不常见的一种情况。

精确抽象要比最佳抽象更加严格，这是因为最佳抽象的定义要求的是两个 $\displaystyle{C\mapsto A}$ 函数相等，而精确抽象的定义要求的是两个 $\displaystyle{A\mapsto C}$ 函数相等。后者要更难一些。

函数抽象的例子

对于函数 $\displaystyle{f = \lambda X.\{x + 1\ |\ x\in X\}}$，$\displaystyle{f'([a, b]) = [-\infty, \infty]}$ 是它的一个可靠抽象，虽然这个抽象毫无意义；$\displaystyle{f'([a, b]) = [a + 1, b + 1]}$ 则是它的精确抽象。

对于函数 $\displaystyle{f(X) = \{2 \times x\ |\ x\in X\}}$ 而言，$\displaystyle{f'([a, b]) = [2\times a, 2\times b]}$ 是它的最佳抽象，但是不是精确抽象。
