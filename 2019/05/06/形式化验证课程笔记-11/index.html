Well-Founded Relations

A binary relation $\prec$ on $R$ is well-founded if for any subset of $R$, there is a "minimal" element, and any "decreasing" sequence from a certain element is finite. (We call $a$ is less than $b$ if $a \prec b$.)

Examples

  • less-than relation on $\mathbb N$ is well-founded, because for any element $n$ in $\mathbb N$, the decreasing sequence $n, n - 1, n - 2, \dots, 0$ is finite.
  • greater-than relation on $\mathbb N$ is not well-founded, because for any element $n$ in $\mathbb N$, there exists an infinite decreasing sequence $n, n + 1, n + 2, \dots$.
  • less-than relation on $\mathbb R^+$ is not well-founded.

Proving Total Correctness

To prove the termination of the program, we can construct a set of states and a relation over it. If the relation is well-founded, and on every basic path, the output state is less than the input state, then the program will terminate eventually.

The running of the program consists of many basic paths. On every basic path, the output state is less than the input state. If the program doesn't terminate, there will be a infinite decreasing sequence on the set of states, which is impossible.

Details

We need to find a ranking function $\delta$ mapping program states to a set $S$ with a well-founded relation $\prec$.

Typically, $S$ is a set of $n$-tuples of natural numbers and the relation $\prec$ is lexicographic version of less-than relation on $\mathbb N\times \dots \times \mathbb N$.

Annotations

$\downarrow$ will be used for the ranking function. Example usages:

  • $\downarrow i$
  • $\downarrow (i + 1, j - 1)$

in which $i$ and $j$ are natural numbers. Typically, a (or some) algebraic expression will be used as the state of the program, such as the length of some array (in this example the ranking function is trivial).

Because a basic path always terminates, we only need to provide the result of applying mapping function on the beginning and ending of the basic paths.
