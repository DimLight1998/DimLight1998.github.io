发表于 2019-05-20 | 更新于 2019-05-21

第十三周

抽象最小不动点的计算

如果完全偏序集 $A$ 上没有无穷递增链，$f'$ 是 $A$ 上的单调函数，$a_0$ 是 $f'$ 的前不动点，那么迭代 $a_{i + 1} = f'(a_i)$ 即可在有限步骤内收敛到 $\text{lfp}_{a_0}f'$。

当 $A$ 上有无穷递增链的时候会比较麻烦，为此引入偏序集 $(A, \sqsubseteq)$ 上的 加宽算子 $\triangledown: A\times A\rightarrow A$，它应当满足：

  • $\forall x, y\in A, x\sqsubseteq x\triangledown y\land y\sqsubseteq x\triangledown y$
  • 对于任意的链 $(x_i)_{i\in \mathbb N}$，按照如下规则构造出来的链 $(y_i)_{i\in \mathbb N}$ 应当是递增的并且 $\exists k\in \mathbb N, y_{k + 1} = y_k$：
    • $y_0 = x_0$
    • $y_{i + 1} = y_i\triangledown x_i$

现在，还是给定完全格 $A$ 和上面的单调函数 $f':A\rightarrow A$，那么下面的递增链：

  • $y_0 = \bot'$
  • $y_{i + 1}=\begin{cases}y_i &\text{if }f'(y_i)\sqsubseteq y_i \\ y_i\triangledown f'(y_i)&\text{otherwise}\end{cases}$

将会在有限步骤内收敛到某个 $y_k$，这个 $y_k$ 是 $\text{lfp}f'$ 的上近似（$\text{lfp}f'\sqsubseteq y_k$）（实际上它是 $f'$ 的后不动点）。可以看到这个算法只是求出了不动点的近似。

（下面的可靠性性质是干嘛的？可能是再从抽象域返回到具体域？）

在进行了上面的操作之后，求出的是后不动点。我们可以继续优化这一结果，使得近似更加精确。为此引入变窄算子 $\triangle:A\times A\rightarrow A$，它应当满足：

  • $\forall x, y\in A, y\sqsubseteq x\rightarrow y\sqsubseteq (x\triangle y)\sqsubseteq x$
  • 对于任意的递减链 $(x_i)_{i\in \mathbb N}$，按照如下规则定义的链 $(y_i)_{i\in \mathbb N}$是递减的并且 $\exists k\in \mathbb N, y_{k + 1} = y_k$：
    • $y_0 = x_0$
    • $y_{i + 1} = y_i \triangle x_{i + 1}$
