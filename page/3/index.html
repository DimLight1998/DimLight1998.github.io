Natural Language Processing & Word Embeddings

Introduction to Word Embeddings

Word Representation

在上一讲中使用了 one-hot 向量对单词进行编码。这样做的弊端是任意两个词之间的距离都是一样的，所有的词都是平权的，实际上并不好。解决的方法是使用高维向量来表示一个词（称为 嵌入），每个分量的取值都可以是一个实数。语义上 接近的单词应当在高维空间中的距离较短。这种嵌入方式可以使用相对于 one-hot 向量更低的维数进行编码，因此被称作是 dense 的。

在理想的情况下，用来表示单词的高维向量的分量应该是具有意义的。例如，第一个分量可以用来指明这个单词是名词的程度，第二个分量可以用来指明这个单词表示一个人类的程度。不过，一般使用高维向量表示单词时每个分量的含义没有这么明确，而是难以解释的。

如果使用了合适的表示手段将词 嵌入 到了高维空间中，然后使用一些降维算法（例如 t-SNE）将其可视化，就可以发现语义上相近的词的分布会更加集中。

Using word embeddings

词嵌入经常和迁移学习结合起来使用。

一般的应用流程是从网上下载已经训练好的词嵌入结果（也可以自己训练），然后将这个嵌入方案应用到新的任务上。此后还可以对嵌入方案进行微调。

词嵌入和将图像编码为向量的思路很类似。区别在于图像的编码对任意图像都可以进行，但是词嵌入只能对词典中存在的词进行。

Properties of word embeddings

词嵌入的一个应用是 analogy reasoning，这和词嵌入的属性有关。

在训练得到词的嵌入方案的过程中可以让词嵌入具有一定的性质。假设要构建这样的一个应用，它可以回答诸如“man 相对于 women 相当于 king 相对于什么”这样的问题。下面使用 $\displaystyle{e_{w}}$ 来表示单词 $\displaystyle{w}$ 的嵌入。要构建的应用相当于找出一个词 d，使得 a 相对于 b 相当于 c 相对于 d，只要让嵌入方案大致满足 $\displaystyle{e_a - e_b \approx e_c - e_d}$ 即可。这样，在给出了 a，b 和 c 后只要求出 $\displaystyle{\arg\max\limits_d\text{similarity}(e_d, e_c + e_b - e_a)}$。

相似性的度量最常用的是使用余弦相似度。度量时应当在进行嵌入的空间中进行度量。使用了 t-SNE 进行降维后的空间中再进行度量时会出问题，因为这种降维破坏了线性性。

Embedding matrix

词嵌入学习到的结果可以表示为一个矩阵 $\displaystyle{\mathbf E}$，行数为嵌入的向量长度，列数为词典大小。每一列是对应词的嵌入结果。

可以使用某个词对应的 one hot 向量 $\displaystyle{\vec o}$ 来取出这个词的嵌入结果，只要计算 $\displaystyle{\mathbf E \vec o}$ 即可。尽管实现上这么做不是高效的。

序列的表示

以文本序列为例，首先准备一个词典，其中包含 <UNK> 表示不在词典中的所有词。每个词都有自己的唯一序号。对于一个序列 $x$，将其中的每个单词都编码为一个 one hot 的向量，其中只有单词在词典中的序号对应的位置是 1，其余位置是零。这样就把一个单词序列编码为了 one hot 向量的序列。

Recurrent Neural Network Model

为何不使用普通网络

  • 普通神经网络在处理序列问题时无法考虑到时间上的局部信息，序列中的每个元素在其看来是平权的。
  • 出于和引入 CNN 一样的原因，使用 RNN 可以学习到如何对临近的元素提取信息，并可以减小参数数目。
  • 序列的长度不是固定的，普通神经网络无法处理这个问题。

递归神经网络

RNN 可以将一个序列映射到另一个序列。在整个序列映射过程中，网络参数保持不变。

对于元素 $x^{\langle t\rangle }$，它对应的 $\hat y^{\langle t\rangle }$ 既取决于输入 $x^{\langle t\rangle }$ 也取决于上次的激活值 $a^{\langle t- 1\rangle }$。在计算过程中，$x^{\langle t\rangle }$ 既产生预测值 $\hat y^{\langle t\rangle }$ 也产生激活值 $a^{\langle t\rangle }$。初始时的 $a^{\langle 0\rangle }$ 可以是零向量，也可以是某个随机值。

第四周

Review FOL

Logical Symbols and Non-logical Symbols

  • Logical symbols: propositional connectives, variables, quantifiers.
  • Non-logical symbols: constant symbols, function symbols, predicate symbols.

Interpretation and Assignment

  • Interpretation fixes values for constants.
  • Assignment fixes values for variables.

First-Order Theory: Motivation

First-order logic is very general but undecidable. By limiting the non-logical symbols, we can have something stronger then propositional logic but still decidable, which is first-order theory. First-order theory is weaker then FOL.

Defining a First-Order Theory

A first-order logic contains:

  • Signature $\Sigma$, a set of non-logic symbols
  • Axioms $\mathcal{A}$, a set of closed formulas over $\Sigma$

$\Sigma$ limits the symbols and $\mathcal A$ gives them semantics.

Formula only containing symbols from $\Sigma$ is called $\Sigma​$-formula.

Terminology

  • T-interpretation/T-model: A tuple of domain and interpretation $(D, I)$ satisfying $\forall A\in \mathcal A, (D, I)\vDash A$. (Domain and interpretation satisfying all axioms.)
  • T-satisfaction: $F$ is T-satisfiable if you can find a T-model $T$ and an assignment $\alpha$ such that $T, \alpha\vDash F$.
  • T-validity: Similar as above, but asks for all T-models and assignments. Written as $T\vDash F$. A theory is the set of all closed formulas that are T-valid.
  • T-entailment: $F_1$ T-entails $F_2$ if $T\vDash F_1\rightarrow F_2$.
  • T-equivalence: Similar as above but asks for $F_1\leftrightarrow F_2$.

Object detection

Detection algorithms

Object Localization

图像输入的任务

对于图像输入，常见的任务有以下几种：

  • Image classification（分类）：对图像进行分类。
  • Classification with localization（带定位的分类）：对图像进行分类并且给出目标的 bounding box。
  • Detection（目标检测）：对图像中的目标给出 bounding box。

其中分类和带定位的分类都只有一个目标对象，而目标检测可能有多个目标对象。

输出的定义

对于带定位的分类，处理手段一般是将图片通过 CNN 然后提取出一个特征，该特征的构成如下：第一个分量 $\displaystyle{p_c}​$ 指示了图片中存在目标的概率（因为这种任务中输入可能是什么目标都没有）；随后的四个分量分别指出了目标在图片中的位置和 bounding box 大小；随后的若干个分量来自于一个 softmax 的输出，指示了目标的分类。因此，如果分类有 $n​$ 个，特征就有 $\displaystyle{n + 5}​$ 个分量。

  • 对于第一个分量，如果接近零表示无目标，此时后续分量没有意义。计算损失函数时记得考虑。
  • 对于后面的四个分量 $\displaystyle{b_x, b_y, b_h, b_w}$，它们的值介于 0 到 1 之间。其中 $\displaystyle{b_x, b_y}$ 指示目标中心点的位置；$\displaystyle{b_h, b_w}$ 指示目标的高度和宽度。位置和长度都是以图片作为基准单位的，图片左上角坐标为 $\displaystyle{(0, 0)}$，右下角为 $\displaystyle{(1, 1)}$；图片的长和宽都定为 1。

在计算损失函数时，一般在 $\displaystyle{p_c = 1}$ 时采用均方误差，$\displaystyle{p_c = 0}$ 时只看算法是否预测对了图像中没有目标。也可以将代表分类的那几个分量单独取出做交叉熵。

Face recognition & Neural style transfer

Face Recognition

What is face recognition?

两个不同任务：

  • Face verification：给定一张图片，判定这张图片是否是某个人
  • Face recognition：给定一张图片和具有很多个人的数据库，判定这张图片是否属于任意一个人并给出是哪个人

显然 recognition 比 verification 要难。

One Shot Learning

实际系统的 face recognition 的主要挑战在于每个人的照片很少，没法像用来分类的网络那样在很多样本上进行训练。这个问题被称为 one shot learning problem。很多时候每类的样本可能只有一个。此外，如果使用普通的 CNN 分类网络，要向数据库中增加一个人就要重新训练整个 CNN，这是不可接受的。

为了解决这个问题需要网络学习一个 similarity function。这个函数接受两张图片并返回它们的相似程度（越低表示越相似）。低于某个阈值时即可认为两张图片上是同一个人。

一种定义相似函数的方式是使用 Siamese 网络架构。首先构建一个 CNN 网络，它将一张图片转换到一个向量表示的特征上。将输入记作 $\vec x$，输出记作 $f(\vec x)​$，相似函数可以定义为

Deep convolutional models: case studies

Case studies

Why look at case studies?

CNN 中很多东西都是模块化的，学习一些 case 有助于自己写出能用的网络。

本讲会学习到 LeNet-5、AlexNet、VGG、ResNet 和 Inception。

Classic Networks

LeNet-5 使用小卷积核（$\displaystyle{3\times 3}$），卷积层后面跟着平均池化层（当时流行平均池化）。卷积层后面跟着池化层是一个常见的模式，重复几次后变被压平，然后进入几层全连接的网络。

AlexNet 比 LeNet-5 要大，但是整体结构基本相似。不同之处在于 AlexNet 中使用了较大的卷积核（$\displaystyle{11\times 11}$），但是随后几层中卷积核大小又慢慢变小了。

VGG 又称 VGG-16，通过小卷积核（全部是 $\displaystyle{3\times3}$，使用 same 填充策略）和窗口大小与步长都为 2 的最大池化提取特征。每层会将高宽缩小一半，通道增加一倍；最后通过几个全连接层。VGG 是一个相对较大的网络，但是结构简单。一个变种是 VGG-19，它是一个更大的网络。

第三周

主题：一阶逻辑。

Motivation

命题逻辑表达能力有限。一阶逻辑可以表达无穷大的集合、对象之间的关系和各种量化关系。

Syntax

  • 一阶逻辑的语法分为两部分——逻辑部分和非逻辑部分。逻辑部分包含逻辑连接词、变量和谓词。通过限制除了逻辑部分之外的部分即非逻辑部分，可以限制一阶逻辑的表达能力。

  • 非逻辑中包括 函数 和 谓词，区别在于函数将对象映射到对象，谓词将对象映射到布尔值。常数可以看做零元常函数；变量可以看做零元谓词。

  • 一阶逻辑中的对象称为 项，包括对象常量、变量和函数。

  • 一阶逻辑中的求值为真值的东西称为 原子 ，包括真值常量、变量和谓词。
  • 原子和它的反合称 文字。

  • 文字称为 公式；公式使用逻辑连接词连接后，或者被量词结合起来使用后也是公式。

  • 在形如 $\forall x.F(x,y)$ 的公式中，$x$ 是 bound 的（被量词修饰），$y$ 是 free 的。$F(x, y)$ 是 $x$ 的 辖域（相当于作用域）。
    没有 free 变量的公式是 closed 的，否则是 open 的；不包含变量的公式是 ground 的。
